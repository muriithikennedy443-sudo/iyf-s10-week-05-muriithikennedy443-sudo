## Project Description
A comprehensive collection of DOM manipulation exercises and projects built during Weeks 9 and 10. Covers selecting elements, traversing the DOM, modifying content, adding/removing elements, event listeners, event objects, bubbling, delegation, form handling, a full To-Do List mini-project, and 5 daily challenges.

## Technologies Used
- HTML5
- CSS3
- JavaScript 

## Features
- DOM selection using all 5 methods (getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll)
- DOM traversal (parent, children, siblings, descendants)
- Modifying text content, HTML content, attributes and styles
- Creating, appending, removing and cloning elements
- Event listeners with mouse, keyboard, form and window events
- Event object properties (target, type, position, preventDefault, stopPropagation)
- Event bubbling and delegation with a delegated task list
- Form validation with real-time feedback and error styling
- Interactive To-Do List with add, toggle, delete, filter and edit
- 5 Daily Challenges: Color Changer, Element Creator, Image Remover, Content Copier, Dark Mode Toggle

## How to Run
1. Clone this repository
2. Open any `index.html` file in your browser

## Lessons Learned
- How the DOM represents HTML as a tree of nodes
- The difference between querySelector and getElementsBy methods
- How event bubbling works and why delegation is more efficient
- How to validate forms in real time using the input event
- How to build a fully functional app using only vanilla JavaScript

## Challenges Faced
- Understanding the difference between textContent and innerHTML and when to use each safely
- Getting event delegation right so that dynamically added elements also respond to clicks
- Managing state (the todos array) and keeping the UI in sync with it on every change
